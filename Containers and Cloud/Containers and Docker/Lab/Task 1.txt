to access it on port 5000:
>> docker run -d --name DEMO -p 5000:80 nginxdemos/hello  

now can be accessed at http://localhost:5000/

to check the logs:
>> docker logs DEMO
>> docker logs DEMO -f  # in runtime

to access the container:
>> docker exec -it DEMO /bin/sh  # where we can linux commands
>> exit

to force delete the container:
>> docker rm DEMO2 -f 

to map to folder:
>> docker run -d --name DEMO -p 5000:80 -v C:\Users\Murti\test:/app  nginxdemos/hello 

to create a volume:
>> docker volume create myvolume 
>> docker volume ls  
>> docker volume inspect myvolume
>> docker run -d --name DEMO -p 5000:80 -v myvolume:/app  nginxdemos/hello 

>> docker volume rm myvolume
