create volume
>> docker run -p 27017:27017 -v data:/etc/mongo -d -e MONGO_INITDB_ROOT_USERNAME=mongoadmin -e MONGO_INITDB_ROOT_PASSWORD=mongoadminpass mongo

to see volume location in the host machine run:
>> docker volume insect data # "data" is the name of the volume
the volume location in the docker is /etc/mongo
