1. upload the app to github:
https://github.com/MurtadaAhmed/terraformtaskboarddemo
2. create the terraform file
3.
>> terraform init
>> terraform fmt
>> terraform validate
>> terraform apply
3. in resource groups >> click on resource group name >> click on app service name >> look at:
- Deployment Center: status of the deployment
- Default domain: the url of the app

4
>> terraform destroy
