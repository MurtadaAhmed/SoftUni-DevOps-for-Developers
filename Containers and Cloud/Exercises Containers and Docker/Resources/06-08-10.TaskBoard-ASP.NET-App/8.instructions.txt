1. create a network
>> docker network create taskboard_network
2. create and connect a sql server container
>> docker run \
-e ACCEPT_EULA=Y \
-e MSSQL_SA_PASSWORD=yourStrongPassword12# \
-p 1433:1433 \
-v sqldata:/var/opt/mssql \
--rm --network taskboard_network \
--name sqlserver \
-d mcr.microsoft.com/mssql/server

3. create and connect a taskboard app container
a. edit tjhe appsettings.json by adding the database information:
"DefaultConnection": "Server=sqlserver;Database=MyDB;User Id=sa;Password=yourStrongPassword12#;MultipleActiveResultSets=true;"
b.build the image again
>> docker build . -f ./TaskBoard.WebApp/Dockerfile -t murtadoc/taskboard_app
c. run the app 
>> docker run -p 5000:80 --rm --name web_app --network taskboard_network murtadoc/taskboard_app
the website can be accessed at:
localhost:5000
4. containers together
inspect the nework to see both containers are within the same network:\
>> docker network inspect taskboard_network
