1. open the solution file using virtual studio
2. right click on TaskBoard.WebApp >> Add >> Docker Support >> Linux
The Dockerfile will be created inside the folder TaskBoard.WebApp
3. build the Image
using CLI (Linux on Windows) navigate to:
/mnt/c/  # this is the C drive in windows
find the parent of the folder TaskBoard.WebApp and run inside it:
>> docker build . -f ./TaskBoard.WebApp/Dockerfile -t murtadoc/t
askboard_app
murtadoc here is the username of the docker hub
4. login to docker hub
docker login
5. push the Image:
docker push murtadoc/taskboard_app
